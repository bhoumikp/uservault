How to Run:

-- Client

- run "npm install" in client dir
- add api server endpoint to .env (refer from .env.example)
- run "npm dev run"

-- Server
-- run "npm install" in server dir
-- add required server envs in .env file (refer from .env.example)

-- For database
-- Use the userVault.sql file and import it on your MySQL server(like MySQL Workbench)
-- give the DB name in /server/.env file, should be the same as your MySQL server DB name

-- run "npm run dev"
